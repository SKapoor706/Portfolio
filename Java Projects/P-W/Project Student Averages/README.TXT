PROJECT TITLE: Student Averages

PURPOSE OF PROJECT: The purpose of this project was to take data from a seperate file and turn that data into an array. Meanwhile, the set of different integers that were a assigned to different strings of names of students were averaged into one single integer, which was the grade of the name of the student it was assigned to.
