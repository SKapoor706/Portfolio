PROJECT TITLE: Overdrawn at the Bank

PURPOSE OF PROJECT: The purpose of this project was to practice the use of functions in real-world programming, such as a bank accounting program. With the use of functions from the "BankAccount" class, the "tester" class was able to proccess efficiently by calling the functions when needed. Moreover, the functions were very useful when the program was interacted with the user by using scanners and multiple loops. 

VERSION or DATE: October 2014

HOW TO START THIS PROJECT: Use the .java files in a java compiler.
